# QRGift

这是一个互联网时代的全能红包项目。

## 项目简介

可以将支付宝当面付红包、账号、卡密等所有礼品类型转成定制的持久化礼品二维码，适合小范围内发放和管理礼物。

过年发红包的时候，可以将支付宝红包转换成好看的礼品二维码(或者做成美丽的二次元周边)，设置好领取时间，然后提前在支付宝中截图上传(预估下时间提前24小时)，然后发给小娃儿，可以自定义一个跳转页面，设计动画，或者问卷！只有答对你是谁才能领取红包，或者写一个自我介绍才能领取红包。然后放到正常的红包里面，有手机的小娃儿也可以领取自用，让他印象深刻的同时(也可以不要上交)，同时可以留作记录以后也能够看到，只要服务不关，他也一直能够看到！

差不多通宵写的，看有没有大佬接力了，大家新年快乐！(我还得上班就是了)

## 注意事项

- 涉及金钱和敏感数据，小白请勿随意部署和使用。
- 请私有部署，使用完之后及时关闭服务。
- 请设置强密码，二维码不要随意分享至互联网，最好仅分享给亲戚朋友或者小范围的人。
- 目前，代码未完成审计，可能存在安全风险，或者较为严重bug，使用前务必考虑好

## 快速开始

### (1) Docker 部署

```bash
git clone git@github.com:source-dream/qr-gift.git
cd /qr-gift/deploy
docker compose up -d --build
```

默认端口是2026，可以自定义 `docker-compose.yml` 中的 `ports` 配置。

### (2) 本地开发（Linux / macOS）

```bash
bash scripts/dev-linux.sh
```

### (3) 本地开发（Windows PowerShell）

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\dev-windows.ps1
```
