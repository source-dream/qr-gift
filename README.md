# QRGift

一个面向小范围自用场景的「礼物二维码管理系统」。

## 快速开始

### (1) Docker 部署

```bash
cd deploy
docker compose up -d --build
```

### (2) 本地开发（Linux / macOS）

```bash
bash scripts/dev-linux.sh
```

### (3) 本地开发（Windows PowerShell）

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\dev-windows.ps1
```
